Sorting Library

Quick Sort

O(n log n)

Divide and conquer strategy — highly efficient partitioning algorithm for rapid sorting.

Merge Sort

O(n log n)

Stable, comparison-based algorithm that divides data into atomic units before merging.

Bubble Sort

O(n²)

Simple comparison sort — adjacent elements are repeatedly swapped if in wrong order.

Heap Sort

O(n log n)

Binary heap based sorting that provides consistent performance and in-place sorting.

Selection Sort

O(n²)

Simple O(n²) comparison sort — repeatedly finds the minimum and moves it to the front.

Insertion Sort

O(n²)

Simple O(n²) insertion sort — efficient for small data sets, builds sorted array one item at a time.

Tim Sort

O(n log n)

Hybrid stable sort derived from merge sort and insertion sort — Python & Java's default.

Intro Sort

O(n log n)

Hybrid sort combining quicksort, heapsort, and insertion sort — used in C++ STL.

Shell Sort

O(n log²n)

Generalized insertion sort using gap sequences — efficient for medium-sized datasets.

PDQ Sort

O(n log n)

Pattern-Defeating Quicksort — optimized quicksort variant resistant to adversarial inputs.

Dual Pivot Quick

O(n log n)

Uses two pivots for three-way partitioning — Java's Arrays.sort() default for primitives.

Block Sort

O(n log n)

In-place stable merge sort using block rearrangement — O(1) extra memory complexity.

Dual Fusion Sort

O(n log n)

Advanced hybrid combining dual-pivot partitioning with adaptive merge strategies.
