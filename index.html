      Quick Sort
      
            
      
              A highly efficient sorting algorithm that uses a divide-and-conquer strategy.
            
      
          
      
          
          
      
            
      
              Best Case
              O(n log n)
              When pivot divides perfectly
            
      
            
      
              Average Case
              O(n log n)
              Typical execution scenario
            
      
            
      
              Worst Case
              O(nÂ²)
              When pivot is smallest/largest
            
      
            
      
              Space Complexity
              O(log n)
              Due to recursive stack calls
            
      
          
      
          
          
      
            
      
              
              
            
      

            
            
      
              
      
                
      
                  
      Characteristics
      
                  
        
                    
      • 
                      check_circle
                      In-place algorithm
                    
        • 
                    
      • 
                      cancel
                      Not stable (relative order not preserved)
                    
        • 
                  
      
                
      
                
      
                  
      Applications
      
                  
      
                    Used in commercial applications for large scale sorting, internal sorting in libraries, and as a
                    component of more complex hybrid algorithms.
